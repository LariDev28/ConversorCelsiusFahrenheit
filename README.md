🌡️ Conversor de Temperatura (Celsius → Fahrenheit)

📚 Sobre o Projeto

Este projeto foi desenvolvido como parte de um exercício proposto pelo professor Gerson Risso, docente da disciplina Programação de Soluções Computacionais, do curso de Ciência da Computação da Universidade Anhembi Morumbi (campus Mooca).

🎯 Objetivo

Elaborar um programa em Java que:

Leia uma temperatura em graus Celsius

Converta esse valor para graus Fahrenheit

Exiba o resultado na tela

🧮 Fórmula Utilizada

A conversão de Celsius para Fahrenheit é feita utilizando a seguinte fórmula:

F=(C×9/5)+32

Onde:

C = temperatura em Celsius

F = temperatura em Fahrenheit

🧠 Arquitetura do Sistema

Assim como no meu projeto de conversão dólar/real esse programa segue um roteiro lógico e direto: ele interpreta a escolha do usuário entre Celsius e Fahrenheit, aplica a fórmula matemática correspondente e entrega o resultado no console.
